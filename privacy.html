Política de Privacidad

Última actualización: 26/03/2026

1. Alcance

Esta política aplica al uso interno del Bot de Discord desarrollado para Lenox HR por Maynar SRL.

2. Datos recopilados

  • ID de usuario de Discord
  • ID de servidor
  • Mensajes enviados al Bot
  • Datos operativos necesarios

3. Finalidad

Los datos se utilizan exclusivamente para procesar solicitudes y mejorar el funcionamiento del Bot.

4. Almacenamiento

Los datos pueden procesarse en tiempo real o almacenarse según necesidades del sistema.

5. Servicios de terceros

El Bot puede utilizar APIs externas (incluyendo servicios de inteligencia artificial) que procesan información.

6. Acceso

El acceso a los datos está restringido a personal autorizado de Maynar SRL.

7. Confidencialidad

Toda la información se considera confidencial dentro del entorno de Lenox HR.

8. Cambios

Esta política puede modificarse en cualquier momento.

9. Contacto

Para temas de privacidad, contactar al área técnica de Maynar SRL.
